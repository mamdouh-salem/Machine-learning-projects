# Dry Beans Classification

## 📌 Overview
This project classifies dry beans into 7 categories using machine learning.

## 📊 Dataset
- 10,000+ samples  
- 16 numerical features  
- 7 classes  

## ⚙️ Techniques Used
- Data Cleaning & Validation  
- Feature Scaling (StandardScaler, MinMaxScaler)  
- Outlier Handling  
- Class Imbalance Handling (SMOTE)  

## 🤖 Models
- Classification Algorithms (Scikit-learn)

## 📈 Evaluation
- Accuracy  
- Precision  
- Recall  
- F1-score  

## 🛠️ Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
