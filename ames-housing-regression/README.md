# Ames Housing Price Prediction

## 📌 Overview
This project predicts house prices using machine learning models on a real-world dataset.

## 📊 Dataset
- 2,900+ records  
- 80+ features  
- Real estate dataset (Ames Housing)

## ⚙️ Techniques Used
- Data Cleaning & Missing Value Handling  
- Feature Engineering  
- One-Hot Encoding & Ordinal Encoding  
- Outlier Detection (IQR Method)  

## 🤖 Models
- Linear Regression  
- Ridge Regression  
- Lasso Regression  

## 📈 Evaluation
- RMSE  
- R² Score  

## 🛠️ Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
